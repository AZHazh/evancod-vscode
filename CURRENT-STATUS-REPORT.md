## 📋 问题清单

### ❌ 问题 1: Extension error: No active session 无法对话

**严重程度**: 🔴 P0 - 阻塞性问题

**根本原因**:
- 前端发送 `ready` 消息后，后端返回 `session.restored` 时 `currentSession` 为 `null`
- 用户点击发送时，`ChatService.sendMessage()` 检查到没有活动会话，抛出错误
- **核心问题**: Webview 启动时没有自动创建会话，依赖用户手动点击"新建会话"

**代码位置**:
- `src/services/webview/WebviewManager.ts:170-176` - 返回 null session
- `src/services/chat/ChatService.ts:192` - 抛出 "No active session" 错误

**解决方案**:
```typescript
// 在 WebviewManager.ts 的 ready 消息处理中
case 'ready':
  // 如果没有活动会话，自动创建一个
  let session = this.chatService.getCurrentSession()
  if (!session) {
    session = this.chatService.createNewSession()
  }
  
  this.postMessage({
    type: 'session.restored',
    data: {
      session,
      sessions: this.chatService.getSessions(),
    },
  })
  break
```
